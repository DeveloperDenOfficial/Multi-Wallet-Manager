\# Multi Wallet Manager Frontend



A modern, lightweight frontend for the Multi Wallet Manager DApp built with Vite and ethers.js.



\## Features



\- Wallet connection (MetaMask, TrustWallet, etc.)

\- Automatic network switching to BSC Testnet

\- USDT spending approval

\- Wallet information display

\- Responsive design

\- Connection persistence



\## Requirements



\- Node.js 16+

\- Ethereum wallet browser extension (MetaMask, TrustWallet, etc.)
